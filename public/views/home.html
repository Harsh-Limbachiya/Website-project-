HTML5 Information

In this website we used HTML5 Topics to get response from the server using the express library.

We made different web pages like home.html, list.html, audio.html, video.html, etc and used the path library to route and display different part of code and webpage.

For Eg.

GET /employee/num
-This route will return a JSON formatted string containing a single employee whose employeeNum property matches the num parameter in the route, ie: http://localhost:8080/employee/5 returns employee 5 and http://localhots:8080/employee/6 returns employee 6 and so on.

GET /audio
-This route simply returns the html code from the audio.html file located within the views directory.
